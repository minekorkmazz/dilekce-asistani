6502 Sayılı Tüketicinin Korunması Hakkındaki Kanun ve kapsamında yayınlanan Yönetmelik gereğince; hiçbir gerekçe göstermeden 14 (On Dört) günlük süre içerisinde Cayma Hakkım bulunmaktadır.
Bu satış sırasında; ayrıca aşağıda bahsedilen usulsüzlüklerin yapıldığı görülmüştür.
- ( ) Sözleşme yapılırken peşinat alınması
- ( ) Cayma Bildirim Belgesi 16 punto ve sözleşmenin ön yüzünde yazılmaması
- ( ) Fatura ve Garanti Belgesi verilmemesi
- ( ) Kendi el yazısı ile sözleşme tarihi atılmaması
- ( ) Mal tesliminden önce senet imzalatılmaması
- ( ) 14 günlük süre içinde her türlü aramaya rağmen cayma hakkının engellenmesi
- ( ) Türkçe tanıtım ve kullanma kılavuzu bulunmaması
- ( ) Bakanlıktan İzin Belgesi sunulmaması
- ( ) Kapıdan Satış Sözleşmesinin Verilmemesi
